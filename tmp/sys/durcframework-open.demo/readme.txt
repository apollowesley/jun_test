durcframework-open项目demo

采用SpringMvc + mybatis + mysql + maven

0. 下载durcframework-open,并在本地mvn install
1. 下载demo,导入到IDE中;
2. 导入数据(Mysql),运行database.sql即可,数据库连接配置在config.properties中
3. 启动tomcat,在durcframework.test.api下面查看例子

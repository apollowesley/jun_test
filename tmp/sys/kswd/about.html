关于本站


本站主要用于个人的常用网址导航，相当于一个网络收藏夹。我们会经常去一些站，或者回将刚了解到的一些有用的网站收藏起来。你可能有过切身的体会，有时候发现了一个很好的网站，但是没有记住它的网址，导致下次怎么也找不到，所以后来就养成了收藏网址的好习惯。很多浏览器都提供了收藏夹的功能，这也能满足很多人的要求，但是每次都要去找收藏夹按钮，然后再找要进入的网站，也不是很方便。受到网址导航网站的启发，我决定做一个自己的导航网站。有人会问，为什么不用现成的网址导航网站呢，上边的内容不是我想要的，而且右一大片一大片的广告。我非常讨厌广告，没看一次广告，就是在无形的为别人创造财富。我创建本项目的目的，就是要制作一个方便易用的“网络收藏夹”。


本站大部分的设计都是借鉴的别人的作品。为了快速构建本项目，我没有选择直接进行开发设计，而是借鉴了一些开源的东西，集百家之长构成了本项目。在这里，我感谢网络上提供免费素材的社区和论坛，以及无私奉献的开发者和设计者，感谢开源精神。本站的主体部分采用的 solookin 的一个英文网址导航项目。页脚的布局采用 alexvgh 的一个页脚设计项目。页面顶部含“旷世的忧伤”字样的logo由U钙网提供免费logo设计。页面顶部的日期时间显示，主体上方的搜索框以及返回顶部和底部按钮都是取自网络素材。


本站顶部的 logo 没有任何含义，就当是美化页面用的。因为鄙人姓旷，曾经读过一本现代诗集《旷世的忧伤》，所以后来就把自己在网络上的很多空间命名为“旷世的忧伤”，旷世谐音“旷氏”。我曾经说，因为喜欢文字，所以把自己读成了悲伤。所以，“旷世的忧伤”这个标签还是挺适合我的。不与夏虫语寒，不与曲人语道，因为生命缺乏言说的条件，这句话说的是我的性格。我们对科技的痴迷, 源自对未来的憧憬，这句话就是我坚持学习科学技术的原因，因为痴迷未来科技。


本项目只适用于个人，没有通用性。你需要拷贝本项目的源码，还需要懂一点 HTML 技术，然后修改 index.html 文件添加你自己的网址。本来想用 PHP 改写本项目的，想做一个后台用来管理收藏的网址，添加直接增删改查的功能。后来因为这样改写需要用到数据库，显得又有那么一点笨重，而且不能直接部署在 github pages 上，再则现在这样已经能够基本满足我自己的需求了，所以也就懒得改了。


欢迎您批评指正本项目，为本项目提供宝贵的建议。本项目托管于 github, gh-pages 分支为最新内容，master 分支我会不定期更新。您可以直接 fork 使用本项目，但未经允许请勿用于任何商业用途。

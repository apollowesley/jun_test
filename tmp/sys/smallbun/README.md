# 项目说明

> 🚀smallbun 大明湖畔最好的企业级开发脚手架

# 技术选型

* 核心框架：Spring Boot
* 安全框架：Spring Security
* 服务端验证：Hibernate Validator
* 持久层框架：MyBatis MyBatis Plus
* 数据库连接池：HikariCP 
...

# 软件要求

* JDK1.8
* MySQL5.7+
* Maven3.0+

# 功能模块

- [x] 用户登录 
- [x] 系统用户
- [x] 系统角色  
- [x] 组织机构  
- [x] 系统权限
- [x] 数据字典
- [x] 登录日志
- [x] 操作日志
- [ ] 通知公告
- [ ] 岗位管理
- [ ] 系统配置
- [ ] 文件上传
- [ ] 在线用户
- [ ] 用户动态权限
- [ ] 用户禁用、锁定后立即生效
- [ ] 用户登出、不单纯清空浏览器token、需要完善后端登出功能
- [ ] 数据权限

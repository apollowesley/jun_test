# 构建发布版本
事先安装好gradle，并配置好gradle的环境变量，然后切换到命令行在，工程的根目录下执行下述指令：

gradle releaseRuntimeZip

发布版本存放目录：build\distributions


# 搭建开发环境

先从github下载工程，然后通过开发工具（eclipse,idea）的gradle插件将genproject导入开发工具即可（eclipse,idea）

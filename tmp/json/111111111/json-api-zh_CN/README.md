现状 Status
------

本标准是目前正在开发中。如需贡献：

1. 获取所有依赖：

    `$ bundle`

1. 搭建本地站点：

    `$ bundle exec rake preview:browser`

1. 编辑 Markdown 文件。
1. 提交更改。
1. 提交 Pull Request。

当更改被合并到 `gh-pages` 分支，本站点将由 [GitHub Pages](http://pages.github.com) 自动构建。

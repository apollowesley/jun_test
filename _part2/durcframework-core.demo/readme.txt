durcframework-core对应的demo
采用SpringMvc + mybatis + mysql + jquery easyui + maven

1. 下载demo,进入项目根目录,运行mvn eclipse:eclipse,然后导入到eclipse中;
2. 导入数据(Mysql),运行database.sql即可,数据库连接配置在server.properties中;
3. 启动tomcat,浏览器输入http://localhost:your_port/your_project_name/index.jsp 实现对学生表的增删改查
4. student/controller包下含有更多例子,请自行体验.
